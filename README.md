# glox

## what
this is a byte-code interpreter project for lox-language.<br>
The original project was implemented in c, here I use golang.<br>

## build glox
go build .<br>


## compile & run the lox file
./glox ./xxx.lox<br>
./glox -D ./xxx.lox<br> // -D means debug
